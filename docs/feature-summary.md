# 機能一覧と概要

## 📋 概要

Ruby on Rails バックエンドと Next.js フロントエンドで実装する機能の一覧です。
インフラはVercelで検討しています。

- **開発体制**: フルタイム1人
- **総工数**: 484時間
- **開発期間**: 約3ヶ月

## 📝 各フェーズの簡単な説明

### フェーズ1: 環境構築・基礎設定
Rails プロジェクトの作成、データベース設定、Docker化、CI/CDパイプラインの構築など、開発の土台を整えます。

### フェーズ2: 認証システム
ユーザー登録、ログイン・ログアウト、パスワードリセット、2要素認証などのセキュリティ基盤を実装します。（SNS認証は削除）

### フェーズ3: プロフィール管理
名前や自己紹介などの基本情報に加え、誕生日、出身地、趣味、MBTIといった拡張プロフィールの登録・更新機能、およびアバター画像のアップロード機能を実装します。

### フェーズ4: 設定管理
プロフィールの公開範囲などのプライバシー設定、パスワード変更などのセキュリティ設定機能を実装します。（通知設定は削除）

### フェーズ5: タロット・診断機能
ユーザーの性格やチームとの相性を診断する機能です。診断結果の保存や履歴表示も含まれます。

### フェーズ6: タグ・共通者機能
プロフィールに登録された「趣味」や「好きな食べ物」などのタグでユーザーを検索し、同じ興味を持つ人を見つける機能です。

### フェーズ7: プロジェクト機能
曼荼羅アートのような画像を登録・表示するシンプルな機能です。（チームの概念を削除）

### フェーズ8: 統合・最適化
開発した機能全体の動作テストを行い、パフォーマンスのボトルネック（N+1クエリなど）を解消します。（工数を削減）

### フェーズ9: デプロイ・本番環境
アプリケーションを **Vercel** と **Heroku** を利用してインターネット上に公開します。（Vercel利用で工数を大幅削減）

### フェーズ10: ドキュメント・リリース準備
APIの仕様書や開発環境の構築手順書を完成させ、最終的なテストとバグ修正を行い、リリース準備を完了します。
