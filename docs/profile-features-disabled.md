## 🔄 復元方法

将来、無効化した機能を有効にする場合は、以下の手順で復元できます：

### ステップ1: コード内のフラグを変更

```typescript
// 修正前（無効化）
{false && (
  <div>チームタスク進捗</div>
)}

// 修正後（有効化）
{true && (
  <div>チームタスク進捗</div>
)}
```

または、`false &&` を完全に削除：

```typescript
// 修正後（有効化）
<div>チームタスク進捗</div>
```

### ステップ2: バックエンドAPIの実装

無効化した機能を有効にする際は、対応するバックエンドAPIも実装する必要があります：

| 機能 | 必要なAPI |
|:---|:---|
| チームタスク進捗 | `GET /api/teams/:id/tasks/progress` |
| 個人課題進捗 | `GET /api/users/:id/goals/progress` |
| 個人実績（エアコース） | `GET /api/users/:id/courses/stats` |
| 学習ページ | `GET /api/courses`, `GET /api/courses/:id` |
