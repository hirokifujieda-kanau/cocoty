# チームビルディング画面 - 機能提案

## 現在実装されている機能
- チーム専用タイムライン（投稿・イベント・アンケート）
- メンバー一覧（管理者・メンバー区別）
- 重要なお知らせ（確認済み/未確認フィルター）
- チーム掲示板
- プロフィール閲覧

---

## 🎯 追加したい機能

### 1. **チーム目標・KPI管理**
#### 機能概要
- チーム全体の目標設定と進捗管理
- 個人の目標とチーム目標の紐付け
- 達成率の可視化（グラフ・チャート）

#### 具体的な機能
- [ ] 月次・四半期・年次目標の設定
- [ ] 目標に対する進捗更新機能
- [ ] 達成度合いのビジュアル表示（円グラフ、棒グラフ）
- [ ] 目標達成時の通知・バッジ付与
- [ ] 目標への「いいね」「応援コメント」機能

#### 期待効果
- チーム全体のベクトル合わせ
- 個人のモチベーション向上
- 定量的な成長の可視化

---

### 2. **1on1 / メンタリング機能**
#### 機能概要
- メンバー同士の1on1スケジュール管理
- メンタリングセッションの記録
- フィードバックの蓄積

#### 具体的な機能
- [ ] 1on1のスケジュール調整
- [ ] セッション前のアジェンダ共有
- [ ] セッション記録（非公開・本人と管理者のみ閲覧）
- [ ] アクションアイテムのトラッキング
- [ ] 定期的なリマインド通知
- [ ] 過去の1on1履歴の検索・振り返り

#### 期待効果
- 継続的なフィードバックループ
- メンバー間の信頼関係構築
- 早期の課題発見と解決

---

## まとめ

これらの機能を段階的に実装することで、単なるコミュニケーションツールから、**チームの成長を支援する総合プラットフォーム**に進化できます。

特に以下の3つの柱を意識：
1. **可視化** - データやスキル、貢献を見える化
2. **つながり** - メンバー間のコミュニケーション促進
3. **成長** - 個人とチームの継続的な成長支援

これにより、心理的安全性が高く、生産性の高いチーム作りを実現します。
