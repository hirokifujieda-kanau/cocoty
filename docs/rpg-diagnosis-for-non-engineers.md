# RPG診断機能 - わかりやすい説明書

**作成日**: 2026年1月19日  
**対象**: 非エンジニアの方（企画担当者・プロジェクトマネージャー等）

---

## 📌 この診断は何をするもの？

**RPG診断**は、ユーザーが簡単な質問に答えることで、自分の性格傾向を5つの「本能」として可視化する機能です。

RPGゲームの職業（フェンサー、ヒーラー、スキーマー、ガンナー、シールダー）に例えて、わかりやすく表現します。

---

## 🎯 5つの本能とは？

### ⚔️ 狩猟本能（フェンサー素質）
- **どんな力？**: 新しい人や環境に積極的に飛び込む力
- **高い人**: 社交的で行動的、チャレンジ精神旺盛
- **低い人**: 慎重で落ち着いている、じっくり考えるタイプ

### 💚 共感本能（ヒーラー素質）
- **どんな力？**: 他人の気持ちを理解して思いやる力
- **高い人**: 優しくて人間関係を大切にする、サポート上手
- **低い人**: 自分の意見をしっかり持つ、周りに流されない

### 🎨 飛躍本能（スキーマー素質）
- **どんな力？**: 創造的なアイデアを生み出す力
- **高い人**: 独創的で芸術的センスがある、発想が豊か
- **低い人**: 現実的で着実、実績を重視する

### 🔧 職人魂（ガンナー素質）
- **どんな力？**: 計画的に準備して丁寧に仕上げる力
- **高い人**: 几帳面で自己管理ができる、責任感が強い
- **低い人**: 臨機応変で柔軟、型にはまらない

### 🛡️ 警戒本能（シールダー素質）
- **どんな力？**: リスクを察知して慎重に判断する力
- **高い人**: 心配性だけどミスが少ない、事前準備が得意
- **低い人**: おおらかでストレスに強い、プレッシャーに動じない

## 📝 診断の流れ

### ステップ1: 質問に答える（12問）
ユーザーは12個の質問に答えます。各質問は5段階で回答します。

**回答の選択肢:**
1. 全く当てはまらない
2. あまり当てはまらない
3. どちらともいえない
4. やや当てはまる
5. 非常に当てはまる

**質問の例:**
- 「知らない人とすぐに会話ができる」
- 「人が快適で幸せかどうか、気になる」
- 「絵画・映像・小説・音楽などの創作活動をしている」
- など

### ステップ2: 性別を選択
性別によって「共感本能」の判定基準が変わります。
- 男性
- 女性

### ステップ3: 結果を見る
5つの本能それぞれについて、**レベル1〜4**で表示されます。

**レベルの意味:**
- **レベル4**: 非常に高い（この本能が最も強い）
- **レベル3**: 高い（この本能が強め）
- **レベル2**: 標準的（平均的）
- **レベル1**: 低い（この本能は弱め）

### ステップ4: 結果の活用
- 自分の強みや特徴を知る
- チームでの役割分担の参考にする
- 他のユーザーと比較して交流のきっかけにする

## 🔢 どうやってレベルが決まるの？（計算の仕組み）

### ステップ1: 質問ごとに点数がつく
各質問の回答に応じて、1〜5点の点数がつきます。

| 回答 | 点数 |
|-----|------|
| 全く当てはまらない | 1点 |
| あまり当てはまらない | 2点 |
| どちらともいえない | 3点 |
| やや当てはまる | 4点 |
| 非常に当てはまる | 5点 |

### ステップ2: 「逆転質問」は点数を反転させる
一部の質問（Q7とQ9）は、**逆転質問**と呼ばれ、点数を反転させます。

**逆転の計算式:** `6 - 回答の点数`

**例：Q7「批判をすることが得意」で「5点（非常に当てはまる）」と答えた場合**
- 通常なら5点
- 逆転処理: `6 - 5 = 1点` に変換
- つまり、批判が得意 = 共感本能が**低い**と判定

**逆転質問の一覧:**
- **Q7**: 人と議論を起こしやすい。批判をすることが得意 → 共感本能の逆転
- **Q9**: ものごとを整理して考えるのが苦手 → 職人魂の逆転

### ステップ3: 本能ごとに関連する質問の点数を合計
各本能には、2〜3個の質問が関連しています。それらの点数を合計します。

#### ⚔️ 狩猟本能（フェンサー）の計算
```
狩猟本能スコア = Q1 + Q6
```
- Q1: 知らない人とすぐに会話ができる（1〜5点）
- Q6: パーティや交流イベントを企画するのが好き（1〜5点）
- **最大10点**

#### 💚 共感本能（ヒーラー）の計算
```
共感本能スコア = Q2 + 逆転(Q7) + Q12
```
- Q2: 人が快適で幸せかどうか、気になる（1〜5点）
- Q7: 批判をすることが得意（1〜5点） → **逆転処理**
- Q12: 他の人の気持ちを思いやり、優先する（1〜5点）
- **最大15点**

#### 🎨 飛躍本能（スキーマー）の計算
```
飛躍本能スコア = Q3 + Q8 + Q11
```
- Q3: 絵画・映像・小説・音楽などの創作活動をしている（1〜5点）
- Q8: 哲学的、精神的なテーマを考えるのが好き（1〜5点）
- Q11: カタカナ語や、むずかしい言葉を使うことが多い（1〜5点）
- **最大15点**

#### 🔧 職人魂（ガンナー）の計算
```
職人魂スコア = Q4 + 逆転(Q9)
```
- Q4: 事前準備は、余裕を持って入念にする方だ（1〜5点）
- Q9: ものごとを整理して考えるのが苦手（1〜5点） → **逆転処理**
- **最大10点**

#### 🛡️ 警戒本能（シールダー）の計算
```
警戒本能スコア = Q5 + Q10
```
- Q5: 気分が落ち込んだり、憂うつになったりする（1〜5点）
- Q10: ストレスを感じたり、不安になったりする（1〜5点）
- **最大10点**

## 📊 計算例：実際にやってみよう

### 例：太郎さん（男性）の場合

**回答内容:**
- Q1（会話ができる）: 4点
- Q2（幸せか気になる）: 3点
- Q3（創作活動）: 2点
- Q4（事前準備）: 5点
- Q5（落ち込む）: 3点
- Q6（イベント企画）: 5点
- Q7（批判が得意）: 4点 → 逆転処理で **2点**（6-4=2）
- Q8（哲学的）: 2点
- Q9（整理が苦手）: 4点 → 逆転処理で **2点**（6-4=2）
- Q10（ストレス）: 2点
- Q11（カタカナ語）: 3点
- Q12（思いやり）: 5点

**計算プロセス:**

1. **狩猟本能** = Q1 + Q6 = 4 + 5 = **9点** → レベル4
2. **共感本能** = Q2 + 逆転(Q7) + Q12 = 3 + 2 + 5 = **10点** → レベル2（男性基準）
3. **飛躍本能** = Q3 + Q8 + Q11 = 2 + 2 + 3 = **7点** → レベル1
4. **職人魂** = Q4 + 逆転(Q9) = 5 + 2 = **7点** → レベル3
5. **警戒本能** = Q5 + Q10 = 3 + 2 = **5点** → レベル2

**太郎さんの診断結果:**
- ⚔️ 狩猟本能: **レベル4**（最も高い！）
- 💚 共感本能: レベル2
- 🎨 飛躍本能: レベル1
- 🔧 職人魂: レベル3
- 🛡️ 警戒本能: レベル2

→ 太郎さんは**フェンサータイプ**（狩猟本能が強い、社交的で行動的）
